# Exam Behavior

Practice exams should mirror the expected real exam format:

- 80 multiple-choice questions
- Four answer options per question
- "All of the above" and "none of the above" style options are allowed sparingly
- 90-minute time limit
- Simple proportional scoring on a 0-20 scale
- Each question is worth 0.25 points
- Wrong and unanswered questions receive 0 points
- Passing score is 9.5 out of 20

The result screen should show the 0-20 exam score prominently and may also show supporting stats such as percentage correct and theme-level performance.

Questions using "all of the above" or "none of the above" should include explanation coverage for each option so the answer is not ambiguous.

In full timed exams, correct answers and explanations should only be shown after the student submits the attempt or the timer expires.

## Practice Sessions

The MVP should also include untimed practice sessions by course theme. Practice sessions do not need to follow the full 80-question blueprint, but they should still use question bank items, explanations, source references, thumbs up/down feedback, and question reporting.

In practice sessions, correct answers and explanations should be shown immediately after each answer.

## Assembly

Each exam attempt should be assembled fresh from the question bank according to the exam blueprint, then saved as a stable snapshot. The selection should favor questions the student has seen less often, while still allowing fallback to repeated questions if a theme does not have enough available questions.
