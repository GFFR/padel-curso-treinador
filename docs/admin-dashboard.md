# Admin Dashboard

The MVP admin dashboard should include:

- Number of student accounts
- Student account list
- Number of completed exam attempts
- Exam attempt list with source scope, blueprint, score, and pass/fail
- Average score per student
- Aggregate performance by course theme
- Reported questions and thumbs-down questions

The admin dashboard should help answer two questions first: whether students are using the system, and which course themes or questions need attention.

Per-question correctness heatmaps and deeper analytics can be added after the core data model is working.
